A simple guide to common HTML & CSS tags

HTML and CSS Tags

What are they and why are they important?

HTML and CSS are the backbone of any site of visual representation. HTML allows you to structure a webpage and CSS allows you to style it as you wish


HTML and CSS "Tags" allow a computer to correctly interpret what you want and are critical to ensure the page displays correctly.


This site will explain some of the most common HTML tags, what they do, and how to use them.


HTML Tags

HTML or Hyper Text Markup Language is made up of multiple tags, with each tag representing something. The most common tags are:


HTML Tag Name What does it mean? What does it do?
<p> A Paragraph. This tag formats any text between the opening tag and the closing as a standard paragraph or main body text.
<title> The title of the webpage. This tag sits inside the head tag and displays the text as the title on the tab of the browser.
<div> a div or division in the HTML. This tag allows code to be divided up in to sections, for easier reading and coding of larger projects.
<h1> A Heading (1). This tag represents a heading, being more prominent to paragraphs. It ranges from h1 to h6.
<img> A image. This tag allows users to input or embed an image in to their site, either from a URL or from a local file.
<table> a Table. This tag allows users to create a table to hold data, much like this one!
